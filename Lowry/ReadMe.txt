Project Lowry

Aim:
Investigate if literature can be used in a more constructivist manner 
than the SMCR fashion than it presently is. Can this lead to enhanced e-learning?

Objectives:
Provide a set of python lessons by package
Provide pragmatic coding techniques

Packages:
Textual analysis
Norel databases
Chat bots
Twitter integration
Email integration
Graphical displays
Web frameworks - Django/GAE
Sense making

Overview:
Using the novel "Small Machines" as the textual basis a simulated character will be created
The character should be able to: 

Read and write from/to social media and mail
Act as a librarian returning pages from the book based on twitter requests 
Engage in conversations with twitter users
Provide a graphical set of analytics
